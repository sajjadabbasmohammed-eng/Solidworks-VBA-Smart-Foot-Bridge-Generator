# Solidworks-VBA-Smart-Foot-Bridge-Generator
SolidWorks 2024-5 VBA API tool that designs and builds a complete steel footbridge from one form. Validates every catalogued beam against five structural checks (bending, shear, von Mises, deflection, frequency), auto-selects the optimum section, then generates parts, assembly, dimensioned drawings, BOM and PDF/DWG .
